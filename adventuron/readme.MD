This is a sub-repository for all my Adventuron scripts.

So what does what?

* adv.bat adds my Adventuron path to my Windows path
* braadv.py checks that Adventuron brackets are balanced. This is helpful when Adventuron's error-checking is obscure.
* b64.py converts a file to base 64.
* cch.py checks for orphan definitions/bracketed text.
* shalf.py creates code for half-right tries and so forth
* tra.py creates a transparent version of a file. -o is probably useful to overwrite, but we don't make overwriting the default
